# 🏐 Volleyball Registration Bot

Телеграм бот для автоматизации записи на волейбольные игры и управления списком участников.

## ✨ Возможности

- **📥 Запись на игру** - Простая регистрация через кнопки
- **📤 Отмена записи** - Легко отписаться если планы изменились
- **📋 Список игроков** - Просмотр всех записавшихся участников
- **⏰ Автоматические уведомления** - Напоминания об оплате и расписании
- **👥 Управление организатором** - Подтверждение проведения игр
- **🔐 Ограничение мест** - Максимум 12 игроков на одну игру

## 🚀 Установка и запуск

### Предварительные требования

- Python 3.8+
- Telegram Bot Token
- Чат ID для администратора, организатора и волейбольного чата

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd volleyball-bot
```

2. Установите зависимости:

```bash
pip install python-telegram-bot nest-asyncio
```

3. Создайте файл `.env` и настройте переменные окружения:

```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
ADMIN_CHAT_ID=your_admin_chat_id
VOLLEYBALL_CHAT_ID=volleyball_chat_id
ORGANIZER_CHAT_ID=organizer_chat_id
PAYMENT_INFORMATION=your_payment_details
```

4. Запустите бота:

```bash
python bot.py
```

## ⚙️ Конфигурация

### Переменные окружения

| Переменная | Описание |
| --- | --- |
| `TELEGRAM_BOT_TOKEN` | Токен вашего Telegram бота |
| `ADMIN_CHAT_ID`	| Chat ID администратора |
| `VOLLEYBALL_CHAT_ID` | Chat ID основного волейбольного чата |
| `ORGANIZER_CHAT_ID`	| Chat ID организатора |
| `PAYMENT_INFORMATION` | Реквизиты для оплаты |

### Настройки в коде

- `DATA_FILE` - путь к файлу хранения данных игроков
- `GAME_DAY` - день недели проведения игр (по умолчанию "воскресенье")
- `MAX_PLAYERS` - максимальное количество игроков (по умолчанию 12)

## 📅 Расписание автоматических уведомлений

Бот автоматически выполняет следующие действия:

- Пятница 11:00 - Закрытие записи и напоминание об оплате.
- Воскресенье 18:00 - Вопрос организатору о том, что игра состоялась и сумме для перевода. В случае подтверждения и вводе суммы, отправка уведомления о необходимости перевести средства.
- Воскресенье 22:00 - Очистка списка игроков после игры.

## 🎮 Использование

### Основные команды

- 📥 Записаться - Зарегистрироваться на ближайшую игру
- 📤 Отписаться - Отменить свою регистрацию
- 📋 Список игроков - Показать всех участников

### Процесс записи

Нажмите кнопку "📥 Записаться"
Подтвердите запись кнопкой "✅ Да"
Получите подтверждение и попадаете в список игроков

## 🗂️ Структура данных

Данные игроков сохраняются в JSON файл со следующей структурой:

```json
[
  {
    "user_id": 123456789,
    "first_name": "Иван",
    "last_name": "Иванов",
    "username": "ivanov"
  }
]
```
